Important HTML Tags in Building Websites

  • <html> — The root element of every HTML page.
  • <head> — Contains meta-information, like title and links to CSS.
  • <title> — Sets the title of the web page shown in the browser tab.
  • <body> — Holds all visible content (text, images, etc.).
  • <h1> to <h6> — Headings, with <h1> being the largest.
  • <p> — Defines a paragraph.
  • <a> — Creates a hyperlink to another page or website.
  • <img> — Displays an image.
  • <ul>, <ol>, <li> — Lists: unordered, ordered, and list items.
  • <div> — Defines a container used for layout and styling.
